# Aartaler Asset Directory

This directory contains the official branding and reference assets
for the Aartaler (ARTLR) token.

The primary logo file referenced in `docs/TOKEN_IDENTITY.md`
is identity-relevant. The authoritative reference is the SHA-256
hash recorded in the Branding Registry of the token identity
document.

Branding updates are governed and may introduce new asset versions.
Historical versions should be preserved in this directory in
accordance with the governance rules defined in
`docs/TOKEN_IDENTITY.md`.
